## RoRBot(Go)

**RoRBot(Go)** is the free and open-source successor to (RoRServerBot)[https://github.com/RigsOfRods/RoRServerBot].

## Getting started

### Supported Systems

| Operating System | Supported | Notes |
| ---------------- | --------- | ----- |
| Ubuntu | Yes | Ubuntu 20.04 LTS and beyond supported |
| RHEL / Fedora Linux / Rocky Linux / AlmaLinux | Yes | RHEL 8 and beyond supported |
| Arch Linux | Maybe | Not tested |
| Debian | Yes | Debian 11 and beyond supported |
| Windows | Yes | Windows 10 and beyond supported |

### System Requirements

To run RoRBot(Go), you will need a Linux or Windows system with a minimum of 1 GB of RAM, 10 GB of disk for logs, and (depending on how many servers you plan to connect to) 100 mbps.

We only support x86_64 and ARM64 architectures.
